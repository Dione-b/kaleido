# Error Codes

Kaleido error codes are public API. Automation may parse the `KALEIDO_*` code, but must not parse human-readable messages.

| Error code | Meaning | Likely cause | Suggested fix | CI handling | Semver stability |
| --- | --- | --- | --- | --- | --- |
| `KALEIDO_CONFIG_NOT_FOUND` | Project config was not found. | Command ran outside a Kaleido project. | Run from the project root or create `kaleido.config.ts`. | Fail CI and fix checkout path or project bootstrap. | Public code; rename/removal is major. |
| `KALEIDO_INVALID_CONFIG` | Project config failed validation. | Missing required config fields or invalid values. | Fix `kaleido.config.ts` schema errors. | Fail CI and correct the committed config. | Public code; rename/removal is major. |
| `KALEIDO_COMMAND_FAILED` | An underlying command failed. | Stellar CLI, Cargo, or another tool returned non-zero. | Re-run the printed command directly for full diagnostics. | Fail CI and inspect the underlying command output. | Public code; rename/removal is major. |
| `KALEIDO_UNEXPECTED_ERROR` | An unexpected non-Kaleido error was normalized. | A dependency or internal path threw an unknown error. | Re-run with the latest output and report the underlying message. | Fail CI and capture logs for triage. | Public code; rename/removal is major. |
| `KALEIDO_STELLAR_CLI_NOT_FOUND` | Kaleido could not find the `stellar` binary. | Stellar CLI is not installed or not in `PATH`. | Install Stellar CLI using the official Stellar setup guide. | Fail CI and install/pin Stellar CLI in the runner image. | Public code; rename/removal is major. |
| `KALEIDO_STELLAR_CLI_VERSION_PARSE_FAILED` | Stellar CLI version output could not be parsed. | `stellar --version` omitted a valid semantic version. | Check the installed Stellar CLI and report the unexpected version output. | Fail CI and pin a Stellar CLI version with parseable output. | Public code; rename/removal is major. |
| `KALEIDO_UNSUPPORTED_CLI_VERSION` | Stellar CLI version is below Kaleido's supported minimum. | Installed Stellar CLI is too old or is a prerelease below the supported range. | Install a supported Stellar CLI version. | Fail CI and upgrade the runner's Stellar CLI. | Public code; rename/removal is major. |
| `KALEIDO_UNTESTED_CLI_VERSION` | Stellar CLI version is newer than Kaleido's tested maximum. | Local or CI runner upgraded Stellar CLI before Kaleido fixtures were updated. | Use a tested Stellar CLI version or pass `--allow-untested-stellar-cli` only for local experiments. | Fail CI; do not override in release jobs. | Public code; rename/removal is major. |
| `KALEIDO_RUST_NOT_FOUND` | Kaleido could not find Rust. | `rustc` is not installed or not in `PATH`. | Install or update Rust. | Fail CI and install Rust before Kaleido commands run. | Public code; rename/removal is major. |
| `KALEIDO_RUST_TARGET_NOT_FOUND` | Required Rust Wasm target is missing. | `wasm32v1-none` has not been installed. | Run `rustup target add wasm32v1-none`. | Fail CI and add the target during runner setup. | Public code; rename/removal is major. |
| `KALEIDO_DEPLOY_FAILED` | Contract deployment failed after command execution started. | Stellar CLI deploy returned an error or deployment output could not be accepted. | Inspect deploy logs, network settings, source account, and artifact path. | Fail CI and retry only after the deploy preconditions are fixed. | Public code; rename/removal is major. |
| `KALEIDO_BUILD_FAILED` | Contract build failed. | Cargo, Rust target setup, or contract source compilation failed. | Inspect Cargo output and fix the contract build. | Fail CI and treat as a build-blocking error. | Public code; rename/removal is major. |
| `KALEIDO_BINDINGS_FAILED` | Binding generation failed. | Stellar CLI binding generation failed or required deployment metadata was missing. | Deploy the contract if needed and rerun binding generation with CLI diagnostics. | Fail CI and do not publish stale generated clients. | Public code; rename/removal is major. |
| `KALEIDO_INVOKE_FAILED` | Contract invocation failed after target validation. | Stellar CLI invocation, RPC simulation, signing, or submission failed. | Check network, source account, contract ID, method, and arguments. | Fail CI and preserve invocation logs for debugging. | Public code; rename/removal is major. |
| `KALEIDO_CONTRACT_NOT_FOUND` | Contract is not configured. | Unknown contract name. | Add the contract to `kaleido.config.ts` or pass a configured name. | Fail CI and correct the command or config. | Public code; rename/removal is major. |
| `KALEIDO_NETWORK_NOT_FOUND` | Network is not configured. | Unknown `--network` value. | Add the network to config or use a configured network. | Fail CI and correct the network selection. | Public code; rename/removal is major. |
| `KALEIDO_ARTIFACT_NOT_FOUND` | Required artifact is missing. | Missing artifacts file, deployed contract record, or Wasm output. | Run `kaleido init`, `kaleido build`, or `kaleido deploy` as appropriate. | Fail CI and ensure prerequisite jobs publish artifacts. | Public code; rename/removal is major. |
| `KALEIDO_ARTIFACT_INVALID` | Artifact file failed validation. | Malformed JSON or unsupported artifact version. | Fix or regenerate `kaleido.artifacts.json`. | Fail CI and regenerate artifacts from a valid Kaleido version. | Public code; rename/removal is major. |
| `KALEIDO_CONTRACT_ID_NOT_FOUND` | Stellar CLI output did not include a contract ID. | CLI output format changed or deploy did not complete. | Check Stellar CLI output and update parser fixtures if needed. | Fail CI and pin/update Stellar CLI compatibility before release. | Public code; rename/removal is major. |
| `KALEIDO_CONTRACT_ARTIFACT_NOT_FOUND` | Client could not resolve a deployed contract artifact. | Contract was not deployed on the selected network, or no explicit `contractId` was passed. | Deploy the contract first or pass `contractId` in the client registration. | Fail CI and make deploy artifacts available to client tests. | Public code; rename/removal is major. |
| `KALEIDO_CONTRACT_DEPENDENCY_NOT_FOUND` | A configured contract dependency could not be resolved. | Dependency name is missing from config or references an unknown contract. | Add the dependency contract to config or fix the dependency name. | Fail CI and fix the dependency graph before deployment. | Public code; rename/removal is major. |
| `KALEIDO_CONTRACT_DEPENDENCY_CYCLE` | Contract dependencies contain a cycle. | Two or more contracts depend on each other directly or indirectly. | Break the cycle or split initialization into a later invoke step. | Fail CI and reject the cyclic dependency graph. | Public code; rename/removal is major. |
| `KALEIDO_CONTRACT_DEPENDENCY_ARTIFACT_NOT_FOUND` | A dependency artifact needed for deploy arguments was missing. | Dependency deployment did not run or artifacts were not persisted. | Deploy dependencies first and ensure artifacts are loaded from the selected network. | Fail CI and require dependency artifact restoration before deploy. | Public code; rename/removal is major. |
| `KALEIDO_DEPLOY_ARG_PLACEHOLDER_INVALID` | A deploy argument placeholder is malformed or unsupported. | Placeholder syntax is invalid or references an unsupported field. | Use the documented placeholder format for dependency contract IDs. | Fail CI and fix the committed deploy argument template. | Public code; rename/removal is major. |
| `KALEIDO_DEPLOY_ARG_PLACEHOLDER_UNRESOLVED` | A deploy argument placeholder could not be resolved. | Dependency artifact data is missing or the current deploy path cannot pass init arguments. | Deploy dependencies first or remove unsupported placeholders. | Fail CI and do not deploy with unresolved placeholders. | Public code; rename/removal is major. |
| `KALEIDO_BINDING_CLIENT_NOT_FOUND` | Generated binding did not export `Client`. | Binding output is missing, stale, or incompatible. | Regenerate bindings with Stellar CLI. | Fail CI and regenerate bindings before publish. | Public code; rename/removal is major. |
| `KALEIDO_BINDING_METHOD_NOT_FOUND` | Requested binding method was not found. | Contract method name is wrong or bindings are stale. | Check the method name or regenerate bindings. | Fail CI and update generated bindings or tests. | Public code; rename/removal is major. |
| `KALEIDO_XDR_BUILD_FAILED` | Client could not build or read transaction XDR. | Binding transaction object is incompatible or lacks `toXDR()`. | Regenerate bindings or provide a compatible binding adapter. | Fail CI and block release until client bindings can build XDR. | Public code; rename/removal is major. |
| `KALEIDO_XDR_PREPARE_FAILED` | Client could not run `prepare()` on the binding transaction (simulation / fee bump path). | RPC simulation failed, binding `prepare()` threw, or the prepared object is invalid. | Check RPC connectivity, simulation errors, and binding compatibility. | Fail CI for deterministic tests; mark external RPC outages separately. | Public code; rename/removal is major. |
| `KALEIDO_XDR_SIGN_FAILED` | Wallet failed to sign the prepared XDR. | Wallet is disconnected, rejected, or on the wrong network. | Connect the wallet and approve the transaction. | In browser CI, fail mocked wallet tests; in manual flows, prompt user retry. | Public code; rename/removal is major. |
| `KALEIDO_XDR_SUBMIT_FAILED` | Client could not submit signed XDR. | RPC rejected the transaction or binding submit method failed. | Check signature, RPC URL, network passphrase, and account state. | Fail CI and preserve RPC response details. | Public code; rename/removal is major. |
| `KALEIDO_XDR_RESULT_FAILED` | Submit succeeded but the binding response was not a recognized result payload. | Response was a non-null object without `txHash`, `transactionHash`, `hash`, or `result` (e.g. `{}` or unrelated fields only). | Inspect `debugRaw` output and align the binding adapter with the expected submit return shape. | Fail CI and capture the raw result payload. | Public code; rename/removal is major. |
| `KALEIDO_WALLET_NOT_CONNECTED` | Wallet public key could not be obtained before building the binding client. | `getPublicKey()` rejected or threw (wallet extension locked, disconnected, or not installed). | Connect the wallet, grant account access, and retry. | Fail automated wallet tests with fixture setup; surface as retryable in UI flows. | Public code; rename/removal is major. |
| `KALEIDO_SOURCE_ACCOUNT_REQUIRED` | Deploy/invoke needs a source account. | `--source` was omitted. | Pass a Stellar CLI identity alias or public account. | Fail CI and provide the configured identity alias secret. | Public code; rename/removal is major. |
| `KALEIDO_UNSAFE_SOURCE_ACCOUNT` | Source account looked like a secret. | Secret key or seed phrase passed as `--source`. | Use an identity alias or public account address. | Fail CI immediately and rotate exposed credentials if logs contain a secret. | Public code; rename/removal is major. |
| `KALEIDO_INVOKE_TARGET_INVALID` | Invoke target did not use `contract.method`. | The invoke target had missing or extra segments. | Use a target such as `counter.increment`. | Fail CI and fix the scripted invoke target. | Public code; rename/removal is major. |
| `KALEIDO_TEMPLATE_NOT_FOUND` | Requested template directory was not found. | Template name is wrong or templates directory is unavailable. | Use a bundled template or set `KALEIDO_TEMPLATES_DIR`. | Fail CI and install or reference the expected template. | Public code; rename/removal is major. |
| `KALEIDO_TEMPLATE_INVALID` | `kaleido.template.json` is not valid JSON or failed manifest schema validation. | Malformed JSON, wrong types, or missing required manifest fields. | Fix `kaleido.template.json` to match the template manifest schema. | Fail CI and block publishing the template package. | Public code; rename/removal is major. |
| `KALEIDO_TEMPLATE_MANIFEST_NOT_FOUND` | Template has no manifest. | Template predates manifest compatibility. | Add `kaleido.template.json`. | Fail CI and require manifests for all bundled templates. | Public code; rename/removal is major. |
| `KALEIDO_TEMPLATE_INCOMPATIBLE` | Template is incompatible with this Kaleido version. | `templateVersion` differs from the current manifest version or `compatibleCore` does not include this `@kaleido/core` release. | Use a compatible template or update `templateVersion` / `compatibleCore` in the manifest. | Fail CI and update template compatibility metadata. | Public code; rename/removal is major. |

Stellar installation guidance: https://developers.stellar.org/docs/build/smart-contracts/getting-started/setup
